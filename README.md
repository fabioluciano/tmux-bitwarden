# tmux-bitwarden

## Requirements
 - Bitwarden CLI
 - skim
 - dasel

## Install
Tmux Plugin Manager (recommended)

1. In your `.tmux.conf` add the plugin to the list.

```
set -g @plugin 'fabioluciano/tmux-bitwarden'
```

2. Type `prefix + I` to install the plugin.

## Key bindings

 - `prefix + b`: list login items in a popup window.
